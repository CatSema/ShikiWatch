# ShikiWatch

[![GitHub release (latest by date)](https://img.shields.io/github/v/release/wheremyfiji/ShikiWatch)](https://github.com/wheremyfiji/ShikiWatch/releases/latest/) [![GitHub all releases](https://img.shields.io/github/downloads/wheremyfiji/ShikiWatch/total?label=All%20Downloads)](https://github.com/wheremyfiji/ShikiWatch/releases/latest/) [![Static Badge](https://img.shields.io/badge/Telegram_Group-2AABEE?logo=telegram&logoColor=white)](https://t.me/shikiwatch)

Неофициальное приложение для [Shikimori](https://shikimori.one/) с возможностью онлайн просмотра аниме.

ShikiWatch объединяет ключевые разделы Shikimori в одном интерфейсе: от поиска и библиотеки до просмотра и управления пользовательским списком. Приложение доступно на Android, Windows и Linux.

### Приложение находится в разработке

### Возможности

- Авторизация через аккаунт **Shikimori** с доступом к персональным данным
- Персональная **библиотека** с быстрым переходом к сохранённым тайтлам
- Гибкий **поиск и фильтры** для подбора аниме по нужным параметрам
- Встроенный **плеер** с выбором источников и продолжением просмотра
- Поддержка [Anime4K](https://github.com/bloc97/Anime4K) — набора open-source алгоритмов апскейлинга и шумоподавления для аниме в реальном времени, что позволяет смотреть аниме в улучшенном качестве без обязательного реэнкодинга в 4K
- **Календарь релизов** с актуальными онгоингами
- **Профиль пользователя** с основной информацией, клубами и историей

> Приложение не является официальным клиентом Shikimori.

## Как использовать приложение

1. Скачайте готовый билд из вкладки [Releases](https://github.com/wheremyfiji/ShikiWatch/releases) и установите его на своё устройство
2. Для **Linux** доступен опциональный способ установки через EPM (особенно актуально для **ALT Linux** и систем с установленным `eepm`/`epm`):
   ```bash
   epm play shikiwatch
   ```
3. Запустите приложение и войдите в свой аккаунт **Shikimori**
4. Наслаждайтесь **онлайн просмотром**, а также удобным доступом к **любимому аниме**
